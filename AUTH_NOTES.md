# Supabase Auth (개발 참고)

연결된 Supabase 프로젝트 Auth에 **이메일 확인이 완료된 시드 계정**이 있습니다.

- **로그인 이메일(아이디)**: `admin@vibe-todo.app`
- **비밀번호**: 저장소에 넣지 않습니다. 프로젝트를 만든 사람에게 별도로 확인하세요.

앱에서는 **로그인** 화면에 위 이메일과 비밀번호를 입력하면 됩니다. (회원가입 없이 사용 가능)

비밀번호를 잊었으면 Supabase 대시보드 → **Authentication → Users**에서 해당 사용자를 재설정하거나 삭제 후 다시 시드하세요.
